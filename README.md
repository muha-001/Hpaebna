# مطعم حبايبنا العالمي (Habaibna Restaurant)

موقع إلكتروني احترافي لمطعم حبايبنا العالمي، يتيح للزبائن تصفح القائمة وطلب الطعام مباشرة عبر الواتساب.

## المميزات
- **نظام طلبات ذكي**: سلة تسوق متكاملة تحسب المجموع ورسوم التوصيل.
- **تكامل واتساب**: إرسال تفاصيل الطلب والعنوان مباشرة إلى إدارة المطعم.
- **تصميم متجاوب**: يعمل بسلاسة على كافة الأجهزة (موبايل، تابلت، كمبيوتر).
- **خرائط تفاعلية**: عرض موقع المطعم بدقة.

## التقنيات المستخدمة
- HTML5 & CSS3
- JavaScript (Vanilla)
- Leaflet.js (للخرائط)
- Font Awesome (للأيقونات)

## هيكلة المشروع
- `index.html`: الملف الأساسي للهيكل.
- `style.css`: كافة التنسيقات والأنماط.
- `script.js`: المنطق البرمجي ونظام السلة.
- `assets/`: مجلد لتخزين الوسائط والصور.

## كيفية التشغيل
فقط قم بفتح ملف `index.html` في أي متصفح حديث.
